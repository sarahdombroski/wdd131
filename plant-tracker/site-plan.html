Sarah Dombroski

WDD 131-2

Overview

Purpose

Plant Tracker is a website that users can add their plants to so they can keep track of them. The tracker will help them know when and how much to water them, as well as a water reminder if the plant needs more water.

Audience

Adults who struggle keeping their plants alive

Dynamic elements

I will use Javascript classes for each of the plants, and then write functions to call and display the information needed. I will also use javascript to search and filter for the plants that they want to see.

Content

Home page

The home page will contain a list of all of the plants. There is a search bar to search by name/description/etc. The user can select one and it will take them to the individual plant page.

Plant Page

The plant page contains all information the user inputed about the plant. It has a picture of the plant, the name, a description, and how often it needs to be watered. It also shows the last time they watered it, if it needs watered, and a button for them to click if they water it.

New Plant

The plant form page will contain a form for the user to fill out to add a plant to their list. They enter all the necessary information and then it will become avaliable on the home and plant pages.
